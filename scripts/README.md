# Scripts — AgentStore

Utility scripts for development and validation.

| Script | Purpose |
|--------|---------|
| `validate_manifests.py` | Check agent/tool JSON manifests are valid |
| `seed_mock_data.py` | Seed backend storage with mock ratings/reviews |

## Usage

```bash
python scripts/validate_manifests.py
python scripts/seed_mock_data.py
```
